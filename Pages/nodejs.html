What is Node.js

  • JavaScript runtime built on Chrome's V8 JavaScript engine
  • JavaScript running on the server
  • Userd to build powerful, fast & scalable web applications
  • Users an event-driven, non-blocking I/O model

Popular modules

  • express - Web development framwork
  • Connect - Extensible HTTP server framwork
  • Socket.io - Server side component for websockets
  • Pug / Jade - Template engine inspired by HAML
  • Mongo / Mongoose - Wrappers to interact with MongoDB
  • Coffe-Script - CoffeeScript compiler
  • Redis - Redis client library
